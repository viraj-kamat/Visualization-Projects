Create a plot from a given dataset

Based on requirements from assignment 1, I have created a bar chart/histogram for a cars dataset.
Select the plots link in the main menu and from the drop down the required variable to plot a chart.

Functionalities
  • Display histogram for numerical values
  • Plot a bar chart for categorical variables
  • On hover of each bar in the chart, display its value and make the bar wider and more prominent
  • For histogram plots enable the ability to increase/decrease bin size. This is done via a slider.

This is a dataset of cars picked from Kaggle.


The following are the links to the dataset:

  • https://www.kaggle.com/CooperUnion/cardataset
  • https://www.kaggle.com/toramky/automobile-dataset
  • https://www.kaggle.com/ljanjughazyan/cars1

The dataset was cleaned using python. Some values which where empty were replaced with 0, in other cases the row was removed alltogether. No specific python module was used for cleaning the data.
